# ゲーム紹介Webページ(特別講義プロジェクト)

## 概要
大学の特別講義(5日間)で制作した、人気ゲームを紹介するWebページです。3人チームで分担し、それぞれが1ページずつ担当しました。
私は「ゼルダの伝説」ページのデザイン・文章・構造をすべて担当し、短期間の中で正確なレイアウトと、ゲームの魅力が伝わる構成を意識して作成しました。

## 構成
- トップページ
- maimai紹介ページ
- ゼルダの伝説紹介ページ(担当)
